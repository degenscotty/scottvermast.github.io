2D GAME ENGINE

During my second year at DAE, I came accross my biggest programming challenge ever. We had to create a fully working 2D game engine starting from a framework our teacher provided. Funny enough, I didn't want to start from that framework and started watching The Cherno on YouTube. Following his tutorials gave me a whole new perspective on how to approach certain problems. To this day I still watch his inspiring videos.

I had a lot of fun creating my own 2D framework but this is still one of the most challenging assignments I have ever had.
Using my engine I created 2 games to show off what it was capable off.

Pengo (game on the left) is an arcade game published by SEGA in 1982.
Bubble Bobble is a platform arcade game published by Taito in 1986.
